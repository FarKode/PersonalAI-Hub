API Setup Guide

Quick tutorial on configuring API keys for PersonalAI Hub

💡 Bring Your Own Key (BYOK): PersonalAI Hub runs completely locally on your device. To enable AI agents, you need to plug in an API key from one of the supported providers. Your key is stored securely using device encryption.

🤖 Groq Recommended · Free & Fast

Groq provides lightning-fast inference for open-source models like Llama 3 and Mixtral. Highly recommended for daily use.

1

Go to the Groq Developer Console and sign in.

2

Navigate to the API Keys section and click Create API Key.

3

Name your key, copy it immediately, and paste it under Groq settings in the app.

🤖 Google Gemini Free Tier Available

Excellent for general context reasoning and complex tasks using Google's generative models.

1

Visit the Google AI Studio API Keys Page.

2

Create a new key (choose a project or create a default one).

3

Copy the generated key and enter it under the Gemini provider in PersonalAI Hub.

🤖 OpenAI Paid (Pay-per-use)

Industry-standard model quality (GPT-4o, GPT-4o-mini). Requires prepaid balance to generate keys.

1

Login to the OpenAI API Platform.

2

Ensure you have added funds under Settings > Billing (minimum $5).

3

Create a new secret key, copy it, and paste it into the OpenAI field in the app.

🤖 OpenRouter Multi-Model Hub

Access hundreds of AI models (including Claude 3.5, Gemini, DeepSeek, and Llama) through a unified API endpoint.

1

Create an account at OpenRouter Console.

2

Go to the Keys page, generate a new API Key, and save the key token.

3

Paste it in PersonalAI Hub under OpenRouter setting.

🤖 Anthropic Claude Paid

Premium intelligence (Claude 3.5 Sonnet) optimized for deep text analysis and coding logic.

1

Register at Anthropic Console.

2

Add billing funds, then go to API Keys and create a key.

3

Copy your key and paste it under the Anthropic config in the app.
