API Setup Guide

DocuMind AI operates on a BYOK (Bring Your Own Key) model. This means you use your own API keys to power the AI. This gives you 100% privacy, no hidden fees, and unlimited potential.

Below is a guide on how to get API keys. We highly recommend Groq for free, lightning-fast usage.

1. Groq (Recommended) FREE & FAST

Groq offers an incredibly fast API and has a generous free tier. It uses open-source models like Llama-3.

Step 1: Visit Groq Console and sign up.
Step 2: Navigate to "API Keys" on the left sidebar.
Step 3: Click "Create API Key". Copy the generated key.
Step 4: Open DocuMind AI -> Settings -> Select "Groq" -> Paste your key.

2. Google Gemini

Google offers a free tier for Gemini API usage through Google AI Studio.

Step 1: Go to Google AI Studio.
Step 2: Sign in with your Google account and click "Create API Key".
Step 3: Copy the key and paste it into DocuMind AI under the "Gemini" provider.

3. OpenAI (ChatGPT)

For professional use, OpenAI's GPT-4o models are available, but require adding a billing method.

Step 1: Visit OpenAI Platform.
Step 2: Log in and go to "API Keys".
Step 3: Create a new secret key and copy it.
Step 4: Paste it into DocuMind AI under the "OpenAI" provider.

4. Anthropic (Claude)

Claude models are excellent for large documents and coding. Requires billing setup.

Step 1: Visit Anthropic Console.
Step 2: Create an account and navigate to API Keys.
Step 3: Create Key, copy it, and paste it into DocuMind AI under "Anthropic".

Need Help?

If you face any issues setting up your API key, feel free to reach out. Remember, your API usage depends on the provider's pricing, but DocuMind AI does not charge you per request.
